# SparkJoy

静的サイトとして動作するエディタです（`index.html` + `app.js` + `styles.css`）。

## ローカルで試す

```bash
python -m http.server 4173 --bind 0.0.0.0
# ブラウザで http://localhost:4173/index.html
```

## 補足

- これは静的配信なので、ビルド手順は不要です。
- `sounds/` 配下もそのまま配信されます。

## スクリーンショット取得メモ（開発向け）

- UI確認時は `4173` 番ポートで配信すると安定します（`8000` は環境によって競合し、白画面/404になる場合があります）。
- 砂時計の見た目確認は `#hourglassCanvas` 要素を直接撮ると判別しやすいです。
